# phantomCards
PHP + React + XHR + MySQL simple Phatom Dust "skills" (cards) self-test web app project.

## Description
Single-page app allowing viewing of and interacting with all skills/cards from the Phantom Dust videogame. Deck builder currently not planned, but possible depending on motivation.

## Why MySQL + PHP?
My current school work teaches both separately, and it appears there may not be an assignment to combine them. Additionally, PHP is typically freely included in hosting plans, and my current host happens to offer MySQL.

## Why React?
The job market, really. I'm not a big enough fan to use it without forcing myself, and this project is an excuse to force myself.

## Why XHR instead of Fetch?
I like how XHR provides a loading status option. Atm the Fetch API doesn't.

## Why Phantom Dust?
I like the game.
